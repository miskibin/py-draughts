py-draughts

Fast, modern draughts library with move generation, validation, PDN support, and AI engine.

Installation

pip install py-draughts

Quick Start

from draughts import Board, AlphaBetaEngine

# Create a board (Standard/International draughts)
board = Board()

# Make moves
board.push_uci("31-27")
board.push_uci("18-22")

# Get legal moves
print(list(board.legal_moves))

# Use the AI engine
engine = AlphaBetaEngine(depth_limit=5)
best_move = engine.get_best_move(board)
board.push(best_move)

Variants

  • draughts.StandardBoard - International draughts (10×10, flying kings)

  • draughts.AmericanBoard - English checkers (8×8, men capture forward only)

  • draughts.FrisianBoard - Frisian draughts (10×10, orthogonal captures)

  • draughts.RussianBoard - Russian draughts (8×8, flying kings)

Board is an alias for StandardBoard.
